Hangman_Game - This app plays a game of hangman with the user, letting them guess values and
indicating whether they're guessed a correct letter or not. Once the user thinks that they've
got an idea of what the word is, they can try guessing the full word. This app also gives the
option of playing a custom game, where the user picks the hidden word. One user could set
the word, then hand off the app to a different user to try guessing.
